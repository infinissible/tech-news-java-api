# Tech-news-java-api

## Description

- A social news site, Just Tech News, builded with Java Spring Boot framework for RestfulAPIs
- Building a Spring Boot API with creating CRUD operations with Spring Data JPA and MySQL database

## Built with

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- Spring Session
- Spring Security
